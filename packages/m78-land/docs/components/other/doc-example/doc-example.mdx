## 代码示例

对最常用的 示例 + 代码 场景封装了全局的 demo 组件

- props.demo 可以传入一个 react element 也可以传入一个组件
- code 可以直接传入代码字符串/一项代码的配置/多项代码的配置数组

<demo
  demo={require("./exam-code.jsx")}
  code={[
    {
      code: require("!!raw-loader!./exam-code.jsx"),
      name: "index.js",
    },
    {
      code: ".red { color: red }",
      name: "style.css",
      language: "css",
    },
  ]}
/>

```js
<demo
  demo={require("./exam-code.jsx")}
  code={[
    {
      code: require("!!raw-loader!./exam-code.jsx"),
      name: 'index.js'
    },
    {
      code: ".red { color: red }",
      name: 'style.css',
      language: 'css'
    },
  ]}
/>

<demo demo={require("./exam-code.jsx")} code={require("!!raw-loader!./exam-code.jsx")}></demo>
```
