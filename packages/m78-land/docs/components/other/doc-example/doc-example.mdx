## 代码示例

对最常用的 示例 + 代码 场景封装了全局的 demo 组件

- props.demo 可以传入一个 react element 也可以传入一个组件
- code 可以直接传入代码字符串/一项代码的配置/多项代码的配置数组

<Demo
  demo={require("./exam-code.jsx")}
  code={[
    {
      code: require("!!raw-loader!./exam-code.jsx"),
      name: "index.js",
    },
    {
      code: ".red { color: red }",
      name: "style.css",
      language: "css",
    },
  ]}
/>

## LiveDemo

自定义组件需要添加到 `packages/m78-land/src/theme/ReactLiveScope/index.js` 才可在试试实时代码块中使用, 但这会导致组件在初始阶段就完全被加载, 可能会影响加载性能, 不建议使用

```jsx live
function Clock(props) {
  const [date, setDate] = useState(new Date());
  useEffect(() => {
    const timerID = setInterval(() => tick(), 1000);

    return function cleanup() {
      clearInterval(timerID);
    };
  });

  function tick() {
    setDate(new Date());
  }

  return (
    <div>
      <h2>It is {date.toLocaleTimeString()}.</h2>
      <Button color="primary">test</Button>
    </div>
  );
}
```
