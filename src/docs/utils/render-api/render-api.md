# RenderApi 渲染 api

有时候我们需要在 react 主实例外渲染组件，例如 Modal、Drawer 等组件，RenderApi 为其提供了便捷的操作，组件只需要遵循一些 api 规则即可。

RenderApi 在独立的包中维护，如果需要独立使用，请安装`@m78/render-api`

具体的使用见[@m78/render-api](https://github.com/m78-core/render-api)
